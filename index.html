Chestgram

Chestgram - CLI клиент Telegram написанный на Python с библиотекой telethon.

Репозиторий 
Список доступных команд
OBJECT — Имя или ID чата, контакта

contacts — Выводит список всех контактов
msg #OBJECT# Текст — Отправить сообщение
last Текст — Отправляет сообщение последнему адресату
history #OBJECT# Кол-во — Выводит историю сообщений, максимум 50
mute #OBJECT# — Отключить вывод сообщений
silent — Включить тихий режим, вывод сообщений только от пользователей
play ID — Проиграть аудио, видео сообщение
exit — Выход из программы

—————————————————————————————————————————————————————————————————————————————— 

Пример использования команд

msg #Lost Fire# Привет!
last Как дела?
history #Lost Fire# 25
mute #960287752#
play 123456
